This repository contains all of the code used to configure openvpn. You can naviguate throught directories and check how it works.

## Usage

### Init openvpn

```bash
make init url=<host url>
```

### Create key

```bash
make key mail=<user_mail>
```

### Start openvpn

```bash
make
```

### Update openvpn

```bash
make update
```
